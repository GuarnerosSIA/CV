Reinforcement learning for controlling continuous systems with uncertain dynamics and restricted states using robust neural dynamic programming

The present page describes the proposition of using Reinforcement Learning (RL) to find the optimal tracking control of a continuous system with uncertain dynamics and subjected to physical constraints. The so-called Shape Memory Alloys (SMAs) that present nonlinearities, hysteresis and need to be protected from overheating fulfill the proposed system characteristics serving as a test system. Adaptive Dynamic Programming (ADP) is used, with an RL technique, for finding the optimal control for a trajectory tracking task. Finally, states restrictions are tackled using Barrier Lyapunov Functions over the optimal control obtained by the ADP. The union of the concepts mentioned above is the main contribution of this work.
